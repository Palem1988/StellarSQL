B+ Tree

Internal Node

+---------------------------+
|P0|K1||P1|K1|| ...  ||Pn|Kn|
+---------------------------+
K: Key Value
P(n-1): Pointer to the page with Kn-1 <= Value < Kn

Leaf Node

+-----------------------------------+
|P0||R0|K0||R1|K1|| ...  ||Rn|Kn||P1|
+-----------------------------------+
K: Key Value
R: Record pointer
P0: Previous page pointer
P1: Next page pointer
